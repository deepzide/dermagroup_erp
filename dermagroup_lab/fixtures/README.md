# Dermagroup Lab - Fixtures

# This directory contains JSON fixtures for customizations
# These will be automatically loaded when the app is installed

## Structure:
# - custom_fields/: Custom Field definitions
# - client_scripts/: Client-side JavaScript
# - server_scripts/: Server-side Python scripts  
# - print_formats/: Custom print templates
# - property_setters/: Property customizations

All fixtures are version controlled and will be synced to the database on app installation/migration.
